# NodeJs常用命令

安装依赖

```bash
npm install
```

指定镜像源

```bash
npm install --registry=https://registry.npm.taobao.org
```

指定安装生产环境依赖

```bash
npm install --registry=https://registry.npm.taobao.org --production
```



安装node-sass

```bash
npm install node-sass --sass_binary_site=https://npm.taobao.org/mirrors/node-sass/
```



运行

开发环境

```bash
npm run dev
```



打包

```bash
npm run build
```
