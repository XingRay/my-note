# 安装Claude Code

1 安装 node

略

2 安装 claude code

```
npm install -g @anthropic-ai/claude-code
```



配置环境变量：

linux

```shell
export ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic
export ANTHROPIC_AUTH_TOKEN=${DEEPSEEK_API_KEY}
export API_TIMEOUT_MS=600000
export ANTHROPIC_MODEL=deepseek-chat
export ANTHROPIC_SMALL_FAST_MODEL=deepseek-chat
export CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1
```

windows

```shell
setx DEEPSEEK_API_KEY "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" /M

setx ANTHROPIC_AUTH_TOKEN "%DEEPSEEK_API_KEY%" /M
setx ANTHROPIC_BASE_URL "https://api.deepseek.com/anthropic" /M
setx API_TIMEOUT_MS "600000" /M
setx ANTHROPIC_MODEL "deepseek-chat" /M
setx ANTHROPIC_SMALL_FAST_MODEL "deepseek-chat" /M
setx CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC "1" /M
setx CLAUDE_CODE_MAX_OUTPUT_TOKENS "64000" /M
```



安装native 版本

下载安装脚本

```
https://claude.ai/install.ps1
```

保存到任意目录， 比如 “E:\download\bootstrap.ps1” 
执行脚本

```
powershell -ExecutionPolicy Bypass -File "E:\download\bootstrap.ps1"
```
